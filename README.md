#  VirtualBox Windows Configuration
## Project Overview
A practical project involving installation and configuration of Windows on VirtualBox. This project covers FTP Server and Web Server configuration as part of Operating Systems course.
## Objectives
- To install and configure Windows on VirtualBox
- To setup and configure FTP Server
- To setup and configure Web Server
- To understand virtualization concepts
##  Key Concepts
- Virtualization
- FTP Server Configuration
- Web Server Configuration
- Network Configuration
- Operating System Installation
##  Working Explanation
Windows was installed on VirtualBox virtual machine. FTP server was configured to allow file transfer between systems. Web server was also configured and tested successfully.
##  Tools & Techniques

- **OS:** Windows
- **Server:** FTP Server, Web Server
- **Concepts:** Virtualization, Networking
